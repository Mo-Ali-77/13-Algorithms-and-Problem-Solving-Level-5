# 13-Algorithms & Problem Solving Level 5 🚀

This repository is dedicated to my journey in mastering **Data Structures and Algorithms** at an advanced level. 
It contains 8 fundamental projects, most of which were implemented from scratch to gain a deep understanding of how these structures function under the hood.
---

## 📋 Repository Content

The repository is organized into **8 core projects**, ranging from low-level data structure implementations to practical software applications:

| # | Project | Description |
|:---|:---|:---|
| 1 | **Double Linked List Class** | A custom implementation of a doubly linked list supporting bidirectional traversal. |
| 2 | **Queue Class** | Building a FIFO (First-In-First-Out) structure based on the Linked List implementation. |
| 3 | **Stack Class** | Building a LIFO (Last-In-First-Out) structure with all standard operations. |
| 4 | **Dynamic Array Class** | A flexible array implementation that handles automatic resizing and memory management. |
| 5 | **Queue Array Class** | Implementation of a Queue using the Dynamic Array class instead of a Linked List. |
| 6 | **Stack Array Class** | Implementation of a Stack using the Dynamic Array class. |
| 7 | **Undo and Redo Project** | A practical system using Stacks to simulate Undo/Redo functionality (e.g., in text editors). |
| 8 | **Queue Line System** | A comprehensive system for managing service lines (e.g., Bank or Hospital ticketing). |

## 📋 محتويات المستودع

يحتوي هذا المستودع على **8 مشاريع** أساسية، تم بناء معظمها من الصفر لفهم كيفية عملها داخلياً:

| # | المشروع | الوصف |
|:---|:---|:---|
| 1 | **Double Linked List Class** | تنفيذ قائمة مرتبطة مزدوجة تدعم التنقل في الاتجاهين. |
| 2 | **Queue Class** | بناء هيكل بيانات "الطابور" (FIFO) باستخدام الـ Linked List. |
| 3 | **Stack Class** | بناء هيكل بيانات "المكدس" (LIFO) مع كامل العمليات الأساسية. |
| 4 | **Dynamic Array Class** | مصفوفة مرنة يتغير حجمها تلقائياً عند إضافة أو حذف العناصر. |
| 5 | **Queue Array Class** | تنفيذ الطابور ولكن باستخدام المصفوفات الديناميكية بدلاً من القوائم. |
| 6 | **Stack Array Class** | تنفيذ المكدس باستخدام المصفوفات الديناميكية. |
| 7 | **Undo and Redo Project** | تطبيق عملي يستخدم الـ Stack لمحاكاة خاصية التراجع والإعادة. |
| 8 | **Queue Line** | نظام لإدارة طابور (مثل طوابير البنوك أو المستشفيات) بشكل متكامل. |
